News Site

A custom News Site created using ReactJS. Allows users to read articles on the site and interact with them by voting on them or by leaving a comment. Users can use filters to help look for articles related to a specific topic that would interest them. Communicates with an API (see next) using Axios to retieve the data to be shown to the user. Created as a solo project.

Github Repo
News Site API

The API backend for the News Site (see previous) created using Node.js and Express. The 'home page' lists all the possible endpoints in the api and the types of requests that can be made to those endpoints, featuring examples of both request and response objects. The databases are managed using PostgreSQL. Created as a solo project.

Treasure Hunt

A game designed around users going outside and using an interactive map, created using the Google Maps APi, to find 'treasures' hidden in the real world. Users can then compete with friends and compare who had found more treasures between each other. Created using NextJS and uses the Google Maps API to generate the map component, and uses an api (see next) to retieve data regarding user treasure information. Created as a team project.

Github Repo
Treasure Hunt API

The API backend for the Treasure Hunt game (see previous) created using Django REST Framework (DRF) to manage and communicate with the databases using PostgreSQL. Uses DRF-Swagger to generate an interactive page of every endpoint available to the API and the functionality to make those requests manually. Created as a team project.
